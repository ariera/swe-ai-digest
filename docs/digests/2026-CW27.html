Better Models: Worse Tools (2026-07-04)

Armin Ronacher investigates a regression in Anthropic's newer models (Opus 4.8, Sonnet 5) that increasingly hallucinate extra, invalid fields in tool call schemas — a behavior absent in older models. He hypothesizes this stems from post-training reinforcement on Claude Code's forgiving, slop-tolerant harness, which silently repairs malformed calls and thus provides no gradient against schema drift. The practical implication for engineers is that tool schemas are no longer neutral contracts: non-canonical schemas may be implicitly penalized by post-training, and strict mode or schema simplification may be necessary mitigations.
