The memory shortage is causing a repricing of consumer electronics (2026-05-22)

The explosive growth of AI data centers is driving memory manufacturers to reallocate wafer capacity toward high-bandwidth memory (HBM) for GPUs — from 2% to an expected 20% by end of 2026 — squeezing supply for consumer DDR/LPDDR RAM. Because HBM requires over 3× the wafer capacity per gigabyte, this structural shift will raise prices for consumer electronics for years, hitting affordable smartphones in emerging markets hardest.

Bliki: Vibe Coding (2026-05-21)

Martin Fowler defines "vibe coding" strictly as LLM-driven development where the programmer never reads the generated code — distinguishing it from "agentic programming," where developers still review and reason about AI-written code. He argues vibe coding is viable only for disposable, low-stakes software due to serious risks around security, correctness, and maintainability. Software engineers should understand this distinction, as conflating the two obscures very different risk profiles and professional responsibilities.
