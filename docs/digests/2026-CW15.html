Anthropic's Project Glasswing - restricting Claude Mythos to security researchers - sounds necessary to me (2026-04-07)

Absurd In Production (2026-04-04)

Armin Ronacher reports on five months of production experience with Absurd, a Postgres-only durable execution system built primarily for AI agent workflows. The core model — tasks decomposed into checkpointed steps, with state entirely in Postgres — held up without fundamental redesign. New additions include decomposed step primitives for conditional logic, task result inspection (enabling parent-child agent workflow patterns), a CLI tool (absurdctl), and a web dashboard. Ronacher notes that each LLM agent iteration becomes a durable step, enabling crash-resilient agent loops. The post also raises a meta-question: in an era where agents can build throwaway implementations, what role does open-source infrastructure tooling play when commercial alternatives abound?
