What is “loop engineering?” (2026-07-14)

Gergely Orosz investigates "loop engineering" — the practice of designing automated systems that repeatedly prompt AI agents toward a goal, rather than manually prompting them — tracing its origins to the "Ralph Wiggum" bash loop technique and its recent adoption as a first-class `/goal` command in Codex, Claude Code, and Hermes. While some engineers find agentic loops valuable for event-driven triggers and overnight coding tasks, others report agent drift, high token costs, and diminishing returns versus human-in-the-loop workflows. Orosz concludes that context engineering may offer broader practical value to most developers than deep loop engineering.
